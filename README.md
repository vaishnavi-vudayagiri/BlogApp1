Backend OverView
This project is the backend structure of a Blog Application developed using:

Node.js
Express.js
MongoDB
Mongoose
The backend provides APIs for:

User authentication
Author operations
Admin operations
Common blog functionalities
It also supports:

JWT authentication
File uploads
Cloudinary integration
Middleware protection
Cookie handling
FrontEnd OverView
This project is the frontend of a MERN Stack Blog Application developed using:\

React JS
Vite
Tailwind CSS
React Router DOM
Zustand
The application provides:

User Authentication
Author Dashboard
Admin Dashboard
Blog Article Management
Protected Routes
Responsive UI
